# 🎓 Chance of Admission Prediction

A Machine Learning project that predicts the probability of getting admission into a university based on student academic performance and profile details.

---

## 📌 Project Overview

This project uses Machine Learning algorithms to estimate a student's chance of admission using parameters like:

- GRE Score
- TOEFL Score
- University Rating
- SOP Strength
- LOR Strength
- CGPA
- Research Experience

The model is trained using historical admission data and predicts the admission probability.
